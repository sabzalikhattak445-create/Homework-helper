# Homework Helper

Next.js project with animated subject logos and AI-powered homework helper using Gemini 2.0 Flash.  

**Deployment (Vercel)**:
1. Upload folder to Vercel.
2. Add `NEXT_GEMINI_API_KEY` in Environment Variables.
3. Deploy. Dependencies install automatically.
